# Work on CIFAR-10 dataset

## Goals

* Validation accuracy > 85%
* Use ResNet18 model

## Results

* Best training accuracy - 97.43%
* Best test accuracy - 88.83%
* Number of epochs - 20
* Parameters - 11,173,962 (for default ResNet18 model)
