## Goals
* Validation accuracy > 99.4%
* Parameters < 20,000
* Epochs < 20
* No fully connected layer

## Model parameters
* Kernel size: 3x3
* Dropout: 15%
* Batch size: 128
* Learning rate: 0.01
* Epochs: 15

## Result
The model reached a test accuracy of 99.46% after 15 epochs.
